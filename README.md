# Context Interview — Teach AI Who You Are in One Session

Stop re-explaining yourself in every AI conversation.

This is a Claude skill that runs a structured 20–30 minute interview about who you are, how you work, and what good output looks like to you — then writes it all into a single `MY-CONTEXT.md` file you can reuse in any AI tool, forever.

## The Problem

Every new chat starts from zero. The AI doesn't know you hate fluffy intros, that you want one recommendation instead of five options, that you're an expert in your field but a beginner in others. So you either re-explain constantly, or you get generic answers.

## What This Does

1. Interviews you across six areas: background, working style, output standards, domains, communication style, and your non-negotiable rules
2. Asks one section at a time, conversationally — not a giant form
3. Writes a lean context file (under 800 words) in **your** voice
4. Includes a built-in privacy guardrail so nothing sensitive ends up in a file you'll paste everywhere

## How to Use It

**Option A — Claude skill (recommended):**
Add this folder to your Claude skills, then say: *"Interview me and build my context file."*

**Option B — No setup at all:**
Copy the contents of `SKILL.md` into any Claude chat and say: *"Follow these instructions and interview me."*

When it's done, put the resulting `MY-CONTEXT.md` wherever your AI reads instructions: Claude Projects, custom instructions, a `CLAUDE.md` file, or just paste it at the start of important chats.

## Why an Interview Beats Writing It Yourself

You don't know what an AI needs to know about you. The interview does. It asks the questions that actually change AI behavior — like "finish this sentence: the output has failed if it ___" — instead of the biographical trivia most people write into profiles.

## Origin

I built the first version of this for myself with a 50-question interview and it changed how useful AI is to me day to day. This is the generalized version so anyone can do the same in one session.
